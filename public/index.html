Referral Bonus — How It Works

Invite friends. Both of you get bonus days!

1
Share Your Room Code
Your unique room code (shown on dashboard) is your referral code. Share it with friends before they create their account.
2
Friend Enters Code on Signup
Your friend enters your room code in the "Referral Code" box during their profile setup.
3
Bonus Applied on First Purchase
The bonus is added to their first subscription only — not given immediately. This ensures fair use.
4
Bonus Days — Tiered by Your Status
You are Premium (Weekly/Monthly) → friend gets +7 days bonus
You are a regular user → friend gets +3 days bonus
You (the referrer) also get the same number of bonus days!
5
One-Time Only — Anti-Abuse
Each user can only use a referral code once. Creating a new email on the same device will not work — we track devices to prevent abuse.
💡 Tip: You can refer unlimited friends — each successful referral extends your subscription!
